# Multi-Scale Information Equilibrium (MSIE) for Robust AI

Reference implementation and reproducibility artifacts for the
MSIE-enhanced UVIF framework. This repository accompanies the paper
introducing cross-scale consistency, equilibrium-driven scale selection,
heterogeneous model disagreement analysis, and adaptive multi-scale
information fusion.

## Repository Contents

-   Validation notebook
-   Mathematical formulations
-   Experimental configurations
-   Figures and generated results
-   Reproducibility scripts
